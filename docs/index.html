🎮 aso bot

Discordでゲームの仲間を簡単に集めよう。
スラッシュコマンド一つで募集を作成・管理できます。

サーバーに追加する

✨ 機能

📋

簡単募集作成

/recruit でゲーム名・人数・日時をその場で設定して投稿。

🔘

ボタンで参加管理

参加・補欠・遅れて参加・途中のみ・辞退をボタン操作で完結。

⏰

DM通知

開始前に自動でDM通知。タイミングは /notify で各自カスタマイズ。

📣

開始メンション

時間になったらチャンネルで参加者全員をメンション。

🔒

ロール制限

特定ロールのメンバーだけに限定した募集を作成できます。

🚫

辞退期限

「開始30分前まで辞退可能」のような辞退期限を設定可能。

📖 使い方

1

サーバーに Bot を追加

上の「サーバーに追加する」ボタンからBotを招待します。必要な権限はメッセージ送信・DM送信のみです。

2

/recruit で募集情報を入力

ゲーム名・最大人数・辞退期限を入力して送信します。最大人数と辞退期限は空欄でもOKです。

3

開始日時を選択

ドロップダウンで日付・時・分を選択して「次へ」を押します。

4

ロール・メンションを選択（任意）

参加を特定ロールに限定したい場合と、募集投稿時にメンションするロールをここで選択。「確定」を押すと募集が投稿されます。

5

メンバーがボタンで参加・開始時刻にメンション

投稿された募集カードのボタンから参加・辞退が可能。時間になると参加者全員がメンションされます。

💬 コマンド一覧

/recruit

遊ぶメンバーを募集します。ゲーム名・最大人数・辞退期限・開始日時・参加ロールを設定して募集カードを投稿。

/notify [タイミング]

開始前のDM通知タイミングを個人設定します。5分前・10分前（デフォルト）・15分前・30分前・60分前・オフから選択。
