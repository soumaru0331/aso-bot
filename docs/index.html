🎮 aso bot

Discordでゲームの仲間を簡単に集めよう。
スラッシュコマンド一つで募集を作成・管理できます。

サーバーに追加する

✨ 機能

📋

募集作成

/recruit コマンドでゲーム名・人数・開始日時をサクッと入力して募集を作成。

✅

参加・辞退ボタン

参加・補欠・遅れて参加・途中のみ・辞退をボタン一つで操作。参加者リストがリアルタイム更新。

⏰

DM通知

開始時刻の前にDMで通知。/notify で自分の通知タイミングをカスタマイズ可能。

📣

開始メンション

開始時刻になると参加者全員をメンション。乗り遅れる心配なし。

🔒

ロール制限

特定のロールを持つメンバーだけに募集を限定することができます。

🚫

辞退期限

「開始30分前まで辞退可能」のような辞退期限を設定して、直前のドタキャンを防止。

📖 使い方

サーバーにBotを追加

上の「サーバーに追加する」ボタンからBotを招待します。

/recruit で募集を作成

ゲーム名・最大人数・辞退期限を入力して送信。続けて開始日時・参加ロール（任意）を選択すると募集が投稿されます。

メンバーがボタンで参加

投稿された募集カードのボタンから参加・辞退が可能。参加者リストは自動更新されます。

開始時刻に自動メンション

設定した時刻になるとチャンネルで参加者全員がメンションされます。DM通知タイミングは /notify で各自設定できます。

💬 コマンド一覧

/recruit

遊ぶメンバーを募集します。ゲーム名・最大人数・辞退期限・開始日時を設定して募集カードを投稿。

/notify [タイミング]

開始前のDM通知タイミングを設定します。5分前・10分前・15分前・30分前・60分前・オフから選択。デフォルトは30分前。
